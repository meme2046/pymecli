# DouZero 斗地主牌型类型值及含义

-   根据 douzero/env/utils.py 文件中的定义
-   这些类型值用于标识玩家出牌的类型,在斗地主游戏中判断牌型的合法性以及比较不同牌型的大小.例如:王炸(5) > 炸弹(4) > 其他普通牌型,相同类型的牌型则比较 rank 值的大小.
-   {4, 5, 8, 9, 10, 11, 12, 13, 14}这些类型值有动画

| 类型值 | 常量名                | 含义                           |
| ------ | --------------------- | ------------------------------ |
| 0      | TYPE_0_PASS           | 不出（过牌）                   |
| 1      | TYPE_1_SINGLE         | 单张                           |
| 2      | TYPE_2_PAIR           | 对子                           |
| 3      | TYPE_3_TRIPLE         | 三张                           |
| 4      | TYPE_4_BOMB           | 炸弹（四张相同的牌）           |
| 5      | TYPE_5_KING_BOMB      | 王炸（大小王）                 |
| 6      | TYPE_6_3_1            | 三带一                         |
| 7      | TYPE_7_3_2            | 三带一对                       |
| 8      | TYPE_8_SERIAL_SINGLE  | 顺子（连续的单张）             |
| 9      | TYPE_9_SERIAL_PAIR    | 连对（连续的对子）             |
| 10     | TYPE_10_SERIAL_TRIPLE | 飞机（连续的三张）             |
| 11     | TYPE_11_SERIAL_3_1    | 飞机带单张                     |
| 12     | TYPE_12_SERIAL_3_2    | 飞机带对子                     |
| 13     | TYPE_13_4_2           | 四带二                         |
| 14     | TYPE_14_4_22          | 四带两对                       |
| 15     | TYPE_15_WRONG         | 错误牌型（不符合任何牌型规则） |
